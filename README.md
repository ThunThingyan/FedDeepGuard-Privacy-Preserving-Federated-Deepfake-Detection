# FedDeepGuard: Privacy-Preserving Federated Deepfake Detection

FedDeepGuard is a federated deepfake detection framework that combines Vision Transformers, frequency-domain analysis, and robust federated aggregation to detect GAN-generated face images while preserving user privacy.

## Overview

Deepfake images generated by modern GANs such as StyleGAN pose significant threats to digital authenticity. Traditional deepfake detectors require centralized data collection, which raises privacy concerns. FedDeepGuard addresses this challenge by training a global model across multiple clients without sharing raw images.

## Key Features

* Federated Learning for privacy-preserving training
* Frequency-Aware Region-Guided Attention (FA-RGA)
* Multi-channel feature fusion:

  * RGB Images
  * Error Level Analysis (ELA)
  * DCT Frequency Features
* Robust aggregation against malicious clients
* Blockchain-audited model aggregation
* Support for Non-IID federated environments

## Novel Contributions

### N1: FedDeepGuard-ViT

A federated Vision Transformer using 7-channel inputs:

* RGB (3 channels)
* ELA (1 channel)
* DCT Frequency Features (3 channels)

### N2: GAN Artifact Poisoning Attack

A novel poisoning attack targeting frequency-domain supervision in federated deepfake detection.

### N3: Frequency Backdoor Attack

An invisible frequency-domain trigger attack designed to bypass deepfake detectors.

### N4: Frequency Consistency Defence

Cross-channel gradient fidelity checking with dynamic malicious client exclusion.

### N5: Blockchain Auditability

Immutable logging of model aggregation rounds using blockchain-based auditing.

### N6: Platform-Based Non-IID Simulation

Realistic federated client distributions simulating social media, news platforms, and user-generated content.

## Dataset

**140k Real and Fake Faces Dataset**

* 70,000 FFHQ Real Faces
* 70,000 StyleGAN Generated Faces
* Resolution: 256 × 256
